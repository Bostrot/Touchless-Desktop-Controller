Touchless-desktop-controller

Control your computer with your hands without touching the mouse. Developed for the Intel RealSense F200

View project on GitHub

Introduction

This project will demonstrate a small part of the possibilities of the RealSense technology. By defining a square on your display it gives the game the location of your hand to move around. It also allows you to control other cool stuff. Please note this project is still in the beta and there could be some bugs.

Installation

To install it on your computer you need the latest version of the Intel RealSense SDK and in some cases Visual Studio 2015 to compile it. I will publish a build version later. You can download the SDK here. If you installed all requirements simply start the application.
